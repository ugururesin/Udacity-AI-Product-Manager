Medical Image Annotation For Pneumonia Detection Model

Instructions

Overview

This instructions are provided for the annotators to label medical images for the Pneumonia detection projects.

Steps

Read the instructions completely before starting to annotate
Determine if the picture has cloudy shades

  • If a cloudy shade that may sign Pneumonia is visible, check the YES box
  • If multiple cloudy shades that may sign Pneumonia, check the YES box
  • If there is no any cloudy shades that may sign Pneumonia, check the NO box

If the picture has fuzzy pixels and/or regions that is not clear to categorize it as cloudy shade, check the UNC box.
